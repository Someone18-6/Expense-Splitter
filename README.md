Expense Splitter
Overview

Expense Splitter is a Java-based application that helps groups manage shared expenses. It automatically calculates balances between users and determines the minimum transactions required to settle debts.

Problem Statement

Managing shared expenses during trips, outings, or group activities can be confusing. Different people pay for different expenses, making it difficult to track who owes whom. This project provides an automated solution to track expenses, calculate balances, and simplify settlements.

Features
Add and manage users
Create expense groups
Record shared expenses
Calculate balances automatically
View who owes whom
Minimize the number of settlement transactions
Store data in a database
Tech Stack
Java
PostgreSQL
JDBC / Spring Boot (depending on what you use)
